# Shadow Donkey Kong - Java Game

The player controls Mario, navigating platforms and ladders, avoiding enemies monkeys, and defeating Donkey Kong using different mechanics across levels.

---

## Gameplay Overview

- **Level 1**
  - Reach Donkey Kong using a hammer
  - Avoid or destroy rolling barrels
- **Level 2**
  - Defeat Donkey Kong by shooting him multiple times with a blaster
  - Intelligent enemies actively move and shoot projectiles
  - Manage limited ammunition and enemy threats

The game features score tracking, time limits, win/loss conditions, and multiple interactive entities.

---

## Key Features

- Object-oriented game architecture
- Two distinct levels with different mechanics
- Gravity-based physics and jumping mechanics
- Collision detection between all entities
- Enemy AI with predefined movement routes
- Item pickup system (hammer and blaster)
- Score calculation based on player actions
- Multiple game screens (start, gameplay, game over)

---

## Game Mechanics

- **Movement**: Left/right movement, ladder climbing, jumping with gravity
- **Physics**: Gravity, terminal velocity, and collision-based platform alignment
- **Combat**:
  - Hammer for close-range invincibility and destruction
  - Blaster with limited bullets for ranged attacks
- **Enemies**:
  - Static and moving hazards
  - Enemies with scripted movement patterns
  - Projectile-based enemy attacks

---

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- Game loop and real-time rendering (BAGEL game engine)
- Collision detection and basic physics simulation
